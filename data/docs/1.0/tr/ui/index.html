Olobase UI

Olobase, Vue.js - Php teknolojileriyle hızlı ve kolay yönetici paneli uygulamaları oluşturmak üzere tasarlanmıştır ve ayrıca kapsamlı özelleştirmelere olanak tanır. Vuetify üzerine kuruludur. Olobase UI aşağıdaki teknolojiler birleşiminden oluşur.

  • Npm
  • Vue.js
  • Vuetify
  • Olobase

Başlarken

  1. Demo uygulama kodlarına gözatın.
    • olobase-demo-ui
    • olobase-demo-php
  2. Kurulum kısmından gereksinimler bölümünü okuyarak sunucu gereksinimleri karşılayın ve demo uygulama için kurulum aşamalarını tamamlayın. Çevre ortamlarınızı yapılandırın.
  3. Dökümentasyonda anlamadığınız bölümler için destek talebi açabilir ya da Github Tartışmalar sayfasından daha önce sorulmuş sorulara göz atabilirsiniz.
