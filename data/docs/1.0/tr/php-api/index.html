Olobase - Php Api

Php API herhangi bir arka uç çerçevesi ile kullabileceği gibi Laminas/Mezzio micro framework üzerine geliştirilmiş PHP dilinde arka uç api desteği de sunar. Olobase php api yazılımı geliştirilirken aşağıdaki teknolojiler temel alınmıştır.

  • Composer
  • Github
  • Open Api (Swagger-Php)
  • Mezzio Framework
  • Olobase/Mezzio
  • Firebase Php-Jwt
  • Php
  • Php Redis

Başlarken

  1. Örnek demo uygulama kodlarına göz atın.
    • olobase-demo-php
  2. Kurulum bölümündeki gereksinimler bölümünü okuyarak sunucu gereksinimlerini karşılayın ve demo uygulaması için kurulum adımlarını tamamlayın. Çevresel ortamlarınızı yapılandırın.
  3. Dokümantasyonda anlamadığınız bölümler için destek talebi açabilir ya da Github Tartışmalar sayfasından daha önce sorulmuş sorulara göz atabilirsiniz.
