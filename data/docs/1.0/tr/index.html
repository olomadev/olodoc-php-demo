Introduction

Olodoc klasörlenmiş markdown dosyalarınızdan belgeler oluşturabilmek için PHP'de geliştirilen açık kaynaklı bir belge oluşturma aracıdır. Bootstrap 5 CSS çerçevesiyle uygulamalarınız için benzersiz şablonlar oluşturmanıza ve özelleştirmenize olanak tanır. Ayrıca sınırsız alt menüler, klasörler, çok dilli belgeler yaratabilir ve belgelerinizi SEO standartlarıyla uyumlu hale getirmek için XML site haritanızı ve meta etiketlerinizi otomatik olarak oluşturur.

Oladoc özelliklerinin tamamı aşağıda listelenmiştir.

Features

  • Kullanımı kolay
  • Markdown dosyalarını destekler
  • Çoklu dil desteği
  • Prism Js kod renklendirme
  • Sınırsız klasör oluşturabilme
  • Bootstrap 5 şablonlama ile uyumlu
  • Yerleşik belge arama
  • Dahili çapa oluşturucu
  • Base64 kodlu görüntü işleme
  • Gezinme çubukları
  • Html meta etiket oluşturma desteği
  • SEO için otomatik sitemap.xml oluşturucu
  • PHP Mezzio Framework ile çalışır
