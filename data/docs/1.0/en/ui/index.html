Olobase UI

Olobase is designed to create fast and easy admin panel applications with Vue.js - Php technologies and also allows extensive customizations. It is built on Vuetify. Olobase UI consists of the following combination of technologies.

  • Npm
  • Vue.js
  • Vuetify
  • Olobase

Getting Started

  1. Check out the demo application codes.
    • olobase-demo-ui
    • olobase-demo-php
  2. Meet the server requirements by reading the requirements section in the Installation section and complete the installation steps for the demo application. Configure your environmental environments.
  3. You can open a support request for sections you do not understand from the documentation, or browse previously asked questions on the Github Discussions page.
